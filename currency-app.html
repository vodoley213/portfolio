Погодное приложение сделано с использованием HTML5, CSS3, JavaScript. По CSS использовано: CSS Flexbox, Медиа-запросы, адаптивный дизайн, Mobile first, CSS-переменные, @font-face. По JavaScript использовано: модули, Fetch API c FXRatesAPI, Object Destructuring, Async/await. В версии приложения с ООП использованы Классы, геттеры-сеттеры, приватные поля. Динамика официальных курсов валют за год сделана с использованием интерактивных графиков на библиотеке визуализации данных Chart.js.
